# Electricity Bill Predictor

A web app to predict and track your monthly electricity bills.

## Features
- User Login
- Dashboard with charts
- Appliance usage tracking
- Bill history
- Bill prediction

## How to Use
1. Open index.html in your browser
2. Sign up or log in
3. Add your appliances and usage
4. View your predicted bill

## Technologies Used
- HTML, CSS, JavaScript
- Firebase (for login and database)
